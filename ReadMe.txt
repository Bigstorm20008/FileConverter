Для использования программы необходимо:
1) Иметь на компьютере SQL Server;
2) Создать базу данных, используя скрипт из папаки Database;
3) Создать источник данных ODBC :
     a) Название источника - BrokersDatasource (можно изменить в файле BrokerDatabase.cpp в функции connect());
     б) Установить режим аутентификации для Windows;
     в) Указать БД по умолчанию - Brokers;
4) В базе данных заполнить данные, пути указанные в БД для исходных файлов должны существовать и содержать файлы .tck,
   пути для сконвертированных файлов, также должны быть созданы перед использованием программы;
5) Из папки lib, установить все программы, соответственно разрядности Вашей системы(32, 64 bit);
6) Запустить exe файл из папки Release;
